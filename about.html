What's This?

ParksPlus is a simple front-end web app that keeps track of Parks you'd like to visit.
It's a basic project built by @MohTanveer It doesn't do much, but sure looks cool 😎

💻 Built With:

👢 Bootstrap 4 Flat UI Theme (Free Version)
💪🏾 Vanilla JS Well, minus Bootstrap's jquery dependency 😡
🚀 Netify + Github Pages Used for hosting & deployment. Here's the repo, please don't make fun of my code, I'm fragile AF.
🗺 Google Places API Used for Autocomplete
🎨 unDraw Super awesome, open source and completely free illustrations
